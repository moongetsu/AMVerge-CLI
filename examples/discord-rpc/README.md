# Discord RPC

Live Discord Rich Presence status updates during detection, export, and merge.
Uses the same CLIENT_ID as the AMVerge desktop app.

---

## Requirements

```bash
pip install amverge[discord]
```

---

## Examples

| file | description |
|---|---|
| [01_basic_rpc.py](01_basic_rpc.py) | connect, update status, disconnect |

---

## Quick Run

```bash
python examples/discord-rpc/01_basic_rpc.py
```
